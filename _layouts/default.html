Community polls

Neovim is a community effort. To reflect this, donations to the Neovim Team on Bountysource will go straight back to the community.

The donations will be used to place bounties on features and issues that are chosen by the community through polls.

Be sure to follow Neovim on Twitter to find out when the polls are announced to help shape the direction of Neovim!

Style guide

The Neovim Community has written a style guide to show the proper form to follow when contributing to the project.

Keeping the code uniform and consistent across the project is important for maintainability. Check out the style guide to see the standards set in place.
